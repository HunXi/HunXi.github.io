更新日期：2017-12-7 05:43:28（初版）

mod名称 项目地址 备注
JourneyMap JourneyMap 小地图

Storage Drawers Storage Drawers 储物抽屉
Chameleon Chameleon 变色龙核心 抽屉模组前置
Extra Utilities2 Extra Utilities 更多实用设备
Inventory Tweaks Inventory Tweaks R键整理
Just Enough Items Just Enough Items (JEI) JEI
Pam’s HarvestCraft Pam’s HarvestCraft 潘马斯农场 一堆吃的
Morpheus（ServerOnly） Morpheus 投票睡觉
Immersive Engineering Immersive Engineering 浸渍工程 水力发电大法好
Immersive Tech Immersive Tech 浸渍工艺的拓展 看图片挺帅的 还加了蒸汽涡轮 听起来就很屌
FTBLib FTBLib FTB Utilities的前置
FTB Utilities FTB Utilities FTB Utilities
Botania Botania 植物魔法 少年，快来学习马猴
Baubles Baubles 添加了首饰物品栏 植物魔法前置
Better Builder’s Wands Better Builder’s Wands （如果加ExtraUtilities2这个mod就不加）更好的建筑权杖 制杖否
RFTools RFTools rf工具
McJtyLib McJtyLib rftools和top的前置 mcjty的mod大多都要这个前置
BiblioCraft BiblioCraft 添加了一堆桌子 板凳 书架 药水架
Thermal Expansion Thermal Expansion 热力膨胀
Thermal Foundation Thermal Foundation 热力基础 热力膨胀的矿物部分
CoFH Core CoFH Core 热力膨胀前置
Thermal Dynamics Thermal Dynamics 热力膨胀管道部分
Applied Energistics 2 Applied Energistics 2 AE2 一起来体验外星科技
Natura Natura 自然模组 添加了一些生态环境 树木 花 之类的
Mantle Mantle 地幔前置 自然模组的前置
Actually Additions Actually Additions 这模组我不知道怎么翻译 实际添加？
AppleSkin（Client） AppleSkin 苹果皮 可以直观的看到每个食物补充的饱食度
Forestry Forestry 没一周目都行耐心的把林业给发展下去 结果每次都没耐心
Iron Backpacks Iron Backpacks 背包
Refined Storage Refined Storage 和AE2重复了 但是想想还是两个都加进来吧
GraveStone Mod GraveStone Mod 墓碑 （你可看见了我的墓志铭）
Environmental Tech Environmental Tech 环境科技 这个mod玩过一次
ValkyrieLib ValkyrieLib 环境科技前置
Redstone Arsenal Redstone Arsenal 红石兵工厂 添加了一些rf驱动的工具 装备什么的
Dark Utilities Dark Utilities 这mod我也不知道怎么翻，也是添加了一些工具 和一些实用的方块
Just Enough Resources Just Enough Resources (JER) jei拓展 可以看到怪物掉落 矿物所在层数 只是看起来很迷 至今不知道那个图是要表达个什么
Industrial Craft Industrial Craft ic2 额 这个古老的mod有人想玩么
BetterFps BetterFps 更好的FPS 据说有一套玄之又玄的算法什么的 比mc原版的算法好 所以有提示FPS的作用
The One Probe The One Probe top 高亮显示 就是你指到哪里上面的那个框框就告诉你这是什么
TOP Addons TOP Addons top的拓展 添加了一些mod方块的信息显示
Fast Leaf Decay（ServerOnly） Fast Leaf Decay 快速树叶掉落 这个只安装在服务器上就行
Wireless Crafting Terminal Wireless Crafting Terminal AE2的拓展 添加了一个屌爆的无线终端
p455w0rd’s Library p455w0rd’s Library 上面那个mod的前置
XNet XNet 物流网络
JEI Bees JEI Bees JEI拓展 对蜜蜂的杂交支持
LootBags LootBags 奖励袋 享受杀怪爆装备的乐趣
Integrated Dynamics Integrated Dynamics 新mod 没玩过 加进来看看
Cyclops Core Cyclops Core 独眼巨人核心 上面那个mod的前置
Common Capabilities Common Capabilities 同样是Integrated Dynamics mod的前置
MalisisDoors MalisisDoors 添加了一堆各种各样的门 还加了个门的动画
MalisisCore MalisisCore 上面那个mod的前置
Astral Sorcery Astral Sorcery 星系魔法 听说加了很多新东西
Neat Neat 显示怪物血量的小mod 没周目都加了
ExtraCells2 ExtraCells2 AE2的拓展 添加了更多容量的储存芯片
The Twilight Forest The Twilight Forest 暮光森岭 作者终于回来填坑了
Buildcraft Buildcraft 这也是一个上古时代的mod 从1.7.10以后就没玩过bc，现在终于升级上来了
MmmMmmMmmMmm MmmMmmMmmMmm 这个mod只加了一个稻草人 用来测试武器的
Fairy Lights Fairy Lights 添加了一些彩灯 圣诞节要到了
Industrial Foregoing Industrial Foregoing 这个也算个新mod吧，MFR的重制版
Refined Storage Addons Refined Storage Addons rs的拓展 添加了一个无线合成终端 还有无限距离的访问 还有个什么背包的样子
Calculator Calculator 运算妖艺 据说这mod发电简直变态
Sonar Core Sonar Core 上面那个mod前置
Silent’s Gems Silent’s Gems 不知道怎么翻译 静谧宝石？添加了一堆的宝石 用来做各种工具 还有魔法 用来替带匠魂
Silent Lib Silent Lib 上面那个mod的前置
Evil Craft EvilCraft 邪恶工艺 （让我们来做一些邪恶的事情吧）这个mod同样需要独眼巨人核心
JEI Integration JEI Integration JEI的拓展
Animania Animania 将原版的动物变得更加真实
CraftStudio API CraftStudio API 上面那个mod的前置
Everlasting Abilities Everlasting Abilities 这个也不知道怎么翻，永恒的能力？ 杀特定的怪会掉落一个水壶一样的东西 花费特定的钱？那个应是钱吧，就能获得许多buff 而且还是永久的（同样需要独眼巨人核心）
Zombie Awareness Zombie Awareness 僵尸意志，加强了僵尸的ai。嗯 也就是僵尸变得更加的丧心病狂了（抖m万岁）我是不是还要把血月也加进来。。。
CoroUtil CoroUtil 上面那个mod的前置
MeeCreeps MeeCreeps 召唤一个小人 能帮你干一些事情
Foam Fix FoamFix 泡沐修复 据说能降低一半的内存使用
Optifine Optifine 高清修复 也是一个优化类模组
TexFix TexFix 修复forge在加载材质的时候的一些问题，能减少1G以上的内存占用具体详情点我
