关于python库pillow的基本使用

什么是pillow

pillow是python平台的一个图像处理库（python3），功能强大，但简单易用。它是pil（python2）的分支，由于pil仅仅支持到python2.7，在加上长时间无人维护，所以在志愿者在pil的基础上开发出了兼容python3的的版本，就是pillow，它不仅支持了python3，还加入了一些新的特性。

pillow的安装

使用pip安装，简单方便。打开控制台（确保pip命令能正确运行），输入下面命令：

pip install pillow

基本使用方法

open

打开一张图片，返回一个图片对象。

show

调用系统默认的图片查看器打开图片

new

创建一张图片
