什么是requests

requests是用python编写 基于urllib库，采用apache2开源协议的http库。比起使用urllib库来说更加方便，可以节约我们大量的时间，完全满足http测试的需求。

requests的安装

requests的安装较为方便直接使用pip命名就能直接安装（这里使用python3）

pip3 install requests
